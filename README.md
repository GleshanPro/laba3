## Запуск

1. Клонирование репозитория
   `git clone git@github.com:GleshanPro/laba3.git`
2. Переход в директорию laba3
   `cd laba3`
3. Создание виртуального окружения
   `uv venv`
4. Активация

##### Linux/Mac:

`.venv/Scripts/Activate`

##### Windows:

`source .venv\Scripts\Activate`

5. Запуск CLI
   `python -m src.main`
6. Запуск тестов
   `uv run pytest`
